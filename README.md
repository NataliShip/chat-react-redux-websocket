# chat-react-redux-websocket

Пример чата на React/Redux с использованием WebSockets

**Установка модулей**

```npm install``` или ```yarn```


**Запуск приложения**

```npm run start``` или ```yarn start```

**Сборка приложения**

```npm run build``` или ```yarn build```

**Запуск тестов**

```npm run test``` или ```yarn test```
